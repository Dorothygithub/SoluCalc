使用说明
1. 选择起始状态：纯碱（全为 A-）如 Tris base、纯酸（全为 HA）如甘氨酸、或等摩尔混合。
2. 大多数实验从纯碱/纯酸粉末配制缓冲液，选择对应起始状态后输入目标 pH。
3. 调节剂浓度支持 mol/L 和 %（质量百分比）两种单位。
4. 缓冲有效范围为 pKa +/- 1，超出范围可能需大量调节剂。
