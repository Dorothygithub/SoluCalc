# 溶液计算助手

一个可离线使用的单文件网页工具，适合上传到 GitHub 作为最小发布版本。

## 目录结构

```text
solution-helper-github/
├─ index.html
├─ README.md
└─ .gitignore
```

## 文件说明

- `index.html`
  主页面文件，直接在浏览器中打开即可使用。
- `README.md`
  仓库说明文件。
- `.gitignore`
  Git 忽略规则，避免提交系统临时文件。

## 使用方式

1. 下载或克隆这个仓库。
2. 直接双击 `index.html`，或在浏览器中打开它。

## 上传到 GitHub

如果你只想上传最小版本，推荐只上传这个目录中的 3 个文件。

## 备注

原始工程目录中还有微信小程序相关文件；这个发布目录不包含那些内容，只保留单文件网页版本。
