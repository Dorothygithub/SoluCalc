# 溶液计算助手

一个面向实验室日常使用的单文件网页计算器，适合快速完成常见溶液相关计算，不依赖后端，也不需要安装。

## 适用场景

这个工具主要用于实验中的基础配液和换算，包括：

- 已知质量和摩尔质量，计算摩尔数
- 已知摩尔数和摩尔质量，计算称量质量
- 按目标浓度和目标体积计算配液用量
- 两种同单位浓度溶液按目标浓度混合
- 按目标 pH 粗略估算缓冲液调节剂加入量
- 查询常见试剂的摩尔质量

## 主要功能

### 1. 摩尔计算

支持质量、摩尔质量、摩尔数三者之间互算。  
输入其中两个量，自动求第三个量。

支持单位：

- 质量：`mg` / `g` / `kg`
- 摩尔数：`μmol` / `mmol` / `mol`

### 2. 溶液配制

根据：

- 化合物摩尔质量
- 目标浓度
- 目标体积

自动计算需要称取的质量，并给出简单配制提示。

### 3. 溶液混合

用于两种已知浓度溶液混合成目标浓度溶液。

当前规则：

- 三个浓度必须使用同一种单位
- 不同单位之间不会自动换算
- 这样可以避免把本来不可直接比较的浓度硬算到一起

### 4. 缓冲液调节

基于 Henderson-Hasselbalch 关系做实验室常用的粗略估算，支持：

- 从纯酸起始
- 从纯碱起始
- 从等摩尔混合起始

可用于快速判断大概需要加入多少酸或碱调节剂。

注意：这部分更适合做预估，不应替代最终实测 pH。

### 5. 试剂库

内置常见试剂摩尔质量，也支持添加自定义试剂，便于重复使用。

## 使用方式

直接在浏览器中打开 `SoluCalc.html` 即可使用。

不需要：

- 安装依赖
- 启动服务
- 联网
